# 更新日志

## 2026-05-06 20:35:00 - 创建中国礼薄应用

### 新增功能
- **礼金记录管理**：支持录入送礼人姓名、金额、礼品内容及日期
- **活动管理**：支持按不同场合（婚礼、百日宴、寿宴、满月酒、乔迁宴、升学宴、开业庆典等）分类管理
- **数据统计**：自动计算总金额、总人数、人均礼金、最高礼金等统计数据
- **导出功能**：支持将单个活动或全部记录导出为 Excel 或 PDF 格式
- **搜索排序**：支持按姓名、礼品、备注搜索，支持按姓名、金额、日期排序

### 界面设计
- 采用中式传统喜庆风格，主色调为中国红配金色
- 使用 Noto Serif SC 中文衬线字体
- 响应式设计，支持移动端和桌面端

### 技术栈
- Next.js 16 + React 19
- Tailwind CSS
- shadcn/ui 组件库
- xlsx + jspdf 导出功能
- localStorage 本地数据存储

## 2026-05-06 20:42:00 - 添加用户认证系统

### 新增功能
- **用户注册**：支持邮箱密码注册，包含姓名信息
- **用户登录**：邮箱密码登录验证
- **会话管理**：登录状态持久化，支持退出登录
- **路由保护**：未登录用户自动跳转到登录页

### 新增页面
- `/login` - 登录页面
- `/register` - 注册页面

### 技术说明
- 演示版本使用 localStorage 存储用户信息
- 生产环境建议使用 Supabase Auth 进行安全认证
